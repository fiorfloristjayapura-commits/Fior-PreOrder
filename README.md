# Scan QR Display

Halaman HTML sederhana untuk display video promosi dengan teks "SCAN QR UNTUK INFO".

## Cara Pakai
1. Upload repo ini ke GitHub.
2. Aktifkan GitHub Pages:
   - Masuk ke Settings
   - Pilih Pages
   - Source: Deploy from branch
   - Branch: main
3. Buka link GitHub Pages yang muncul.

## Ganti Video
Edit bagian ini di `index.html`:

```html
<source src="LINK_VIDEO.mp4" type="video/mp4">
